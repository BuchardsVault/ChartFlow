About ChartFlow

ChartFlow is a declarative mini‑language powered by TextX & Python. Type plain‑English commands like show "AAPL" from 2023‑01‑01 to 2023‑12‑31 as candlestick and the interpreter fetches historical prices and renders publication‑ready charts or data tables – all in seconds and with zero Pandas boilerplate.

Key Features

  • Sentence‑like syntax: Quickly describe what you want, not how to code it.
  • Built‑in chart types: candlestick, line, bar, and OHLC out of the box.
  • Lightweight options: Specify asset symbol, date range, and chart style in a single line.
  • Instant CLI feedback: Run python3 chartflow.py program.cf and see tables or plots pop up immediately.
  • Pure‑Python install: No C‑extensions; works on any machine with pip install chartflow.
  • Extensible grammar: Modify the TextX file to add keywords or new statements in minutes.
