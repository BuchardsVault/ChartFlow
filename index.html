About ChartFlow

ChartFlow is a lightweight programming language built with TextX & Python that lets analysts pull market data, run quick computations, and render professional‑looking charts in a couple of lines.

Key Features

  • Intuitive syntax – show "TSLA" from 2023‑01‑01 to 2023‑12‑31
  • Multiple chart types – candlestick, line, bar, OHLC
  • Variable binding & expressions for quick calculations
  • Extensible grammar written in TextX (see diagram below)
  • Command‑line interpreter for instant feedback
